CDAYOLOv9 is a crater detection algorithm developed by fine-tuning YOLOv9 on a custom multi-illumination LROC NAC image dataset.
Please note that currently only  8-bit GeoTIFF format imagery is supported.
Usage:
CDAYOLOV9 --image-list G:/XieBin/MYProgram/CDAYOLOV9/testData/list.txt

Testing Hardware Configuration:
GPU: NVIDIA GeForce RTX 3090 (24 GB VRAM)
CPU: Intel Xeon Silver 4216 @ 2.10 GHz
System RAM: 64 GB
